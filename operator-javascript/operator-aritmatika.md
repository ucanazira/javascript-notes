## Operator Aritmatika âž•

Operator untuk operasi matematika dasar seperti penjumlahan, pengurangan, perkalian, pembagian, dan modulus.

| Operator | Deskripsi           | Contoh   | Hasil |
| -------- | ------------------- | -------- | ----- |
| `+`      | Penjumlahan         | `5 + 3`  | 8     |
| `-`      | Pengurangan         | `5 - 3`  | 2     |
| `*`      | Perkalian           | `5 * 3`  | 15    |
| `/`      | Pembagian           | `10 / 2` | 5     |
| `%`      | Modulus (sisa bagi) | `10 % 3` | 1     |
| `**`     | Pangkat             | `2 ** 3` | 8     |

**Contoh kode:**

```js
let a = 10;
let b = 3;

console.log(a + b); // 13
console.log(a - b); // 7
console.log(a * b); // 30
console.log(a / b); // 3.3333...
console.log(a % b); // 1
console.log(2 ** 4); // 16
```
